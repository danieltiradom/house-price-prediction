# Predicción del Precio de Vivienda

## Objetivo
El objetivo de este proyecto es elaborar un modelo de Machine Learning simple para predecir los precios de las viviendas usando Regresión Lineal

Este proyecto fue creado con el propósito de practicar el análisis de datos, la selección de características, entrenar un modelo y evaluarlo.

---

## Dataset

El dataset contiene información sobre la vivienda como:
- Habitaciones
- Baños
- Pies cuadrados (de construcción y terreno)
- Pisos
- Frente al mar
- Vista
- Condición
- Año de construcción
- Año de renovación
- Precio (variable objetivo)

La meta es predecir el **precio** de la casa basado en estas características.

---

## Metodología

Este proyecto sigue los siguientes pasos:

1. Exploración de datos (EDA)
2. Selección de características
3. División de los datos en Entrenamiento/Pruebas (80% entrenamiento, 20% pruebas)
4. Entrenamiento del modelo de regresión lineal
5. Evaluación del modelo utilizando:
     -Error Absoluto Medio (MAE)
     -Error Cuadrático Medio (MSE)
     -Puntuación R²

---

## Rendimiento del modelo

- **MAE:** ~$167,000
- **Puntuación R²:** ~0.51

El modelo explica aproximadamente el 51% de la variación de los precios de la vivienda

En promedio, las predicciones difieren de los precios reales en alrededor de $167,000.

---

- Pandas
- NumPy
- Matplotlib
- Scikit-learn

# Housing Price Prediction

## Objective

The objective of this project is to build a simple machine learning model to predict housing prices using Linear Regression.  

This project was created with the purpose of practicing data analysis, feature selection, training a model and evaluating it.

---

## Dataset

The dataset contains housing information such as:

- Bedrooms
- Bathrooms
- Square footage (living area and lot)
- Floors
- Waterfront
- View
- Condition
- Year built
- Year renovated
- Price (target variable)

The goal is to predict the **price** of a house based on its features.

---

## Methodology

The project follows these steps:

1. Data exploration (EDA)
2. Feature selection
3. Train/Test split (80% training, 20% testing)
4. Linear Regression model training
5. Model evaluation using:
   - Mean Absolute Error (MAE)
   - Mean Squared Error (MSE)
   - R² Score

---

## Model Performance

- **MAE:** ~$167,000  
- **R² Score:** ~0.51  

The model explains approximately 51% of the variance in housing prices.

On average, predictions differ from actual prices by around $167,000.
